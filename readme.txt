== Description ==

WP Team Display is a free WordPress team plugin that allows to easily create and manage teams. You can create WordPress Team, Staff, groups members section on your WordPress site using our WordPress team plugin.

[__Live Pro Demo__](https://codecanyon.net/item/fancy-wordpress-team-showcase/25151565) | [__Demo__](#screenshots) 

<h2>You can display single teams as multiple responsive, you can also showcase all teams in various styles.</h2>

<h2>If you are looking to use WordPress default author plugin then <a href="https://wordpress.org/plugins/author-profiles/">Click here</a> which will create WordPress authors on your WordPress website using the widget area section</h2>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-team-free.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget sections to show the profiles author.
